What was done

For this assignment I was tasked with performing a weather check on 500+ cities across the globe. To do this I designed a program to randomly generate latitiude and longitude combinations, then to grab the city nearest to that geoloaction. After grabbing more than 500 cities with this method, I accessed the application programing interface or API OpenWeatherMap API, linked in the above navbar. With this list of city names being passed through another program that fetched weather data from this API, I created multiple lists and merged them into a DataFrame to use for visualizations.

Using Pandas and Matplotlib I was able to visualize the data fetched into singluar table called a DataFrame or DF. Using the data is this DF I created the 4 distinct visualizations included in this web page; Humidity, Cloudiness, Wind Speed, and Temerature as Y varaibles compared across the X varaible latitude. Hence the Website name.
